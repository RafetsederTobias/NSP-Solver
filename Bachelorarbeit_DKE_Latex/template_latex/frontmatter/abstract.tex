\chapter*{Abstract}

Creating duty schedules for nursing staff is a complex task that must account for both organisational requirements and individual employee preferences. This thesis presents the development of an ASP-based system for the automated generation and short-notice adjustment of schedules in a hospital, implemented as a full-stack web application using Angular, FastAPI, and PostgreSQL.
The optimisation logic is handled by the ASP solver Clingo, which models hard constraints such as minimum staffing and qualification requirements, as well as soft constraints in the form of fair workload distribution across wards. The system further supports a rescheduling procedure for short-notice absences and the proactive computation of alternative plans. The thesis demonstrates that ASP provides a viable foundation for practical scheduling software in the healthcare sector.
